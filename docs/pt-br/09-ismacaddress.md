Formatos de endereços MAC:

```php
// separado por dois pontos
new IsMacAddress('00:1A:2B:3C:4D:5E');

// separado por hífen
new IsMacAddress('00-1A-2B-3C-4D-5E');

// em caixa alta
new IsMacAddress('00:1A:2B:3C:4D:5E');

// em caixa baixa
new IsMacAddress('00:1a:2b:3c:4d:5e');
```
