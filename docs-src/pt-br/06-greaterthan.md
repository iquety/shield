# GreaterThan

--page-nav--

O número é maior do que o esperado.

```php
// string numérica 9 é maior que 8
new GreaterThan('9', 8);

// inteiro 9 é maior que 8
new GreaterThan(9, 8);

// decimal 9.1 é maior que 9
new GreaterThan(9.1, 9);

// decimal 9.9 é maior que 9.8
new GreaterThan(9.9, 9.8);
```
