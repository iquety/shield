# LessThan

O número é menor que o esperado.

```php
// string numérica 8 é menor que 9
new LessThan('8', 9);


// inteiro 8 é menor que 9
new LessThan(8, 9);

// decimal 9.1 é menor que 9.2
new LessThan(9.1, 9.2);
```
