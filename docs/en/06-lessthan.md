# LessThan

-- | -- | --

The number is smaller than expected.

```php
// numeric string 8 is smaller than 9
new LessThan('8', 9);

// integer 8 is smaller than 9
new LessThan(8, 9);

// decimal 9.1 is smaller than 9.2
new LessThan(9.1, 9.2);
```
