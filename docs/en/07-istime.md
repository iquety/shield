# IsTime

[◂ IsDateTime](07-isdatetime.md) | [Documentation Summary](index.md) | [IsIp ▸](08-isip.md)
-- | -- | --

Hours, minutes and seconds format with 24-hour limit:

```php
// minimum hours allowed
new IsTime('00:00:00');

// maximum hours allowed
new IsTime('23:59:59');
```
